Hand Gesture Volume and Brightness Controller

Python OpenCV MediaPipe Human-Machine Interface

An AI-powered computer vision system built in Python that utilizes real-time hand gesture tracking to control Windows system volume and monitor brightness seamlessly. By using a standard webcam, the application translates physical distance metrics between your fingers into hardware execution commands.

  • Dual Hand Independence: Dedicated execution mapping based on hand classification:
    • Right Hand: Controls the system master volume.
    • Left Hand: Controls the monitor display brightness.
  • Intuitive Precision Control: Adjusts levels dynamically by increasing or decreasing the physical distance between your Thumb Tip and Index Finger Tip. Bring your Thumb and Index finger apart to raise levels; pinch them close together to lower levels.
  • Graceful Resource Termination: Safely releases localized webcam threads and closes video windows immediately when clicking on the camera view and pressing the "q" key on your keyboard.
